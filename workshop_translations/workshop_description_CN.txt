[h1]扩展你的视野[/h1]

Project Zomboid 把许多本来你的角色能看到的东西藏在墙壁、树木和上层楼之后。PeekAView 扩展了三个渲染通道，让你的屏幕更接近角色的视野。三个独立功能：

[list]
[*][b]墙壁切除[/b]：墙壁在更远的距离被切除，使转角和门框在你接近时更早显示出后方的内容。
[*][b]树木淡化[/b]：你视野中的树木变为半透明。在视线可达之处发现树列后的僵尸；提早看见障碍物，让驾驶更安全。
[*][b]楼梯视图[/b]：当你正在攀爬楼梯时上层楼会被渲染，让你在到达之前就能看到上面等待的东西。
[/list]

与原版相同的规则。无 X 光透视，无透明墙壁，无敌人轮廓。你角色的视线仍然决定什么是真正可见的。

[h1]设置[/h1]
每项功能都是独立的。你可以三项一起开启、只开启需要的一项，或介于其间。每项功能都有自己的启用开关，墙壁和树木淡化的范围可在选项菜单中调整。

选项 → Mod → Peek a View，四个区段：通用、墙壁切除、树木淡化、楼梯视图。[b]F8[/b] 在游戏内切换主开关（绿色/红色光晕确认；可在 选项 → 按键设置 中的 [noparse][PeekAView][/noparse] 重新绑定）。

[h1]常见问题[/h1]

[b]与 Staircast Workshop Mod 是否冲突？[/b]
不冲突。如果两个都已订阅，PeekAView 会在运行时检测到 Staircast，并让出自己的楼梯视图，两者可以共同运行。

[b]在多人游戏中能用吗？[/b]
可以，仅客户端。每个客户端也需要安装 ZombieBuddy。

[b]我在游戏会话中将 PeekAView 从 Mod 列表中禁用了，但效果仍然显示。怎么办？[/b]
Project Zomboid 在不同存档加载之间将 Mod 代码保留在内存中，因此在会话中将 PeekAView 从 Mod 列表中禁用并不会自行停止它。三种清除方式：（1）在 Mod 自身的选项中关闭（立即生效）；（2）重新加载存档（PeekAView 会在每次存档加载时检查活动 Mod 列表，如果不再存在则自我停用）；或（3）重启 PZ。这是已经报告过的框架级限制。

[b]会影响性能吗？[/b]
多个缓存机制让运行时开销接近原版。站立和步行是开销最小的路径。如果你的硬件吃力，请降低范围滑块。
