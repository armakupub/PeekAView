[h1]扩展你的视野[/h1]

Project Zomboid 把许多本来你的角色能看到的东西藏在墙壁、树木和上层楼之后。PeekAView 扩展了三个渲染通道，让你的屏幕更接近角色的视野。三个独立功能：

[list]
[*][b]墙壁切除[/b]：墙壁在更远的距离被切除，使转角和门框在你接近时更早显示出后方的内容。
[*][b]树木淡化[/b]：你视野中的树木变为半透明。在视线可达之处发现树列后的僵尸；提早看见障碍物，让驾驶更安全。
[*][b]楼梯视图[/b]：当你正在攀爬楼梯时上层楼会被渲染，让你在到达之前就能看到上面等待的东西。
[/list]

与原版相同的规则。无 X 光透视，无透明墙壁，无敌人轮廓。你角色的视线仍然决定什么是真正可见的。

[h1]设置[/h1]
每项功能都是独立的。你可以三项一起开启、只开启需要的一项，或介于其间。每项功能都有自己的启用开关，墙壁和树木淡化的范围可在选项菜单中调整。

选项 → Mod → Peek a View，四个区段：通用、墙壁切除、树木淡化、楼梯视图。[b]F8[/b] 在游戏内切换主开关（绿色/红色光晕确认；可在 选项 → 按键设置 中的 [noparse][PeekAView][/noparse] 重新绑定）。

[h1]常见问题[/h1]

[b]与 Staircast Workshop Mod 是否冲突？[/b]
不冲突。如果两个都已订阅，PeekAView 会在运行时检测到 Staircast，并让出自己的楼梯视图，两者可以共同运行。

[b]在多人游戏中能用吗？[/b]
可以，仅客户端。每个客户端也需要安装 ZombieBuddy。

[b]"Fix B42 Wall-Hiding Bug" 这个选项具体做什么？[/b]
本 Mod 更广的墙壁切除会放大一个原版 B42 的怪癖：上层楼的墙壁在玩家自建结构旁会从某些视角消失，留下墙体空洞。该选项（默认开启，仅在户外）抑制这种效果。代价：常常会在外墙的某一格上残留一片地板贴图，并且贴在原版墙边的玩家自建平台在外立面上比平常更显眼。我将其作为整体沉浸感的较小之恶。如果你的存档在朝向相机一侧的原版房屋上没有大量玩家自建结构，或者你根本不使用墙壁切除功能，可以关闭该选项以获得更接近原版的渲染效果。

[b]我在 ZombieBuddy 的批准对话框中不小心拒绝了 PeekAView。能撤销吗？[/b]
ZombieBuddy 把每条决定保存在 [noparse]%USERPROFILE%\.zombie_buddy\mod_approvals.json[/noparse]。打开该文件，找到含 [noparse]"id": "PeekAView"[/noparse] 且 [noparse]"decision": false[/noparse] 的代码块，把 [noparse]false[/noparse] 改为 [noparse]true[/noparse]，保存，重启 PZ。如果觉得编辑 JSON 有风险，直接删除这些 PeekAView 代码块即可，ZombieBuddy 会在下次加载存档时重新询问。

[b]会影响性能吗？[/b]
多个缓存机制让运行时开销接近原版。站立和步行是开销最小的路径。如果你的硬件吃力，请降低范围滑块。
