僵尸不再玩捉迷藏。

原版 PZ 中，墙壁和屋顶只在你角色周围约 5 格范围内淡出——屏幕前的你看到的比你角色实际能感知的更少。一个靠在房屋墙边或躲在高栅栏后的僵尸对你完全不可见，即使你的角色正直视着它。[b]Peek a View[/b] 弥补了这个差距：房屋、阻挡视线的栅栏，以及现在的树木，都会从更远处开始淡出——呈现在你屏幕上的内容（大致）与你角色真正能看到的一致。

[h1]此 Mod 的功能[/h1]
[list]
[*][b]墙壁切除[/b] — 将原版约 5 格的淡出范围扩展到最多 20 格。可配置。
[*][b]树木淡化[/b] — 角色视线上的树木变得半透明，使其后方的僵尸可见（不是树内的僵尸——非 X 光透视）。独立的范围滑块，最大 25 格。
[*]每项功能都有自己的驾驶速度门限——在车里可以关闭一项而保留另一项激活（例如：在更高速度下保持树木淡化以发现路边障碍物，关闭墙壁切除以减少屏幕干扰）。
[*]可选的仅在瞄准时启用模式——将整个 Mod 限制为仅在你瞄准武器时运行。
[*]无透明墙壁，无 X 光透视——与原版相同的淡出机制，只是从更远处触发。你看到角色看到的内容。
[*]还修复了 Build 42 的渲染错误：玩家建造的结构会导致附近的原版墙壁消失。
[/list]

[h1]设置[/h1]
所有选项位于 选项 → Mod → Peek a View，分为三个区段（通用、墙壁切除、树木淡化）。[b]F8[/b] 切换 Mod 开关（绿/红光晕确认），可在 选项 → 按键设置 中的 [noparse][PeekAView][/noparse] 重新绑定。

[h1]权衡[/h1]
PeekAView 有意改变原版 PZ 的视觉体验——值得了解你将面对什么。
[list]
[*]并非 TIS 所设计的外观。扩展的范围是对开发者所交付视觉风格的有意偏离。如果你偏好原版外观，此 Mod 不适合你。
[*]在密集的城市街区中，移动时建筑物可能会更快地出现和消失——可能会令人分心。
[*]在高速驾驶时，每棵树的淡入动画跟不上场景进入屏幕的速度——因此树木淡化的速度门限默认设置为 50 km/h。
[*]每帧的成本很小（缓存使其接近原版），但不是零——在低端硬件上，将范围减少几格会有帮助。
[/list]

[h1]为什么使用 ZombieBuddy？[/h1]
这是一个 Java Mod。使用标准 Lua Mod 无法改变已编译 PZ 方法的行为，使用 ZombieBuddy 可以使补丁在 PZ 的小版本更新中持续工作。一次性安装——一旦 ZombieBuddy 在你的 Mod 列表中，任何基于 ZombieBuddy 的 Mod 都可以无需进一步配置即可添加。

[h1]源代码与透明度[/h1]
PeekAView 和 ZombieBuddy 都采用 MIT 许可证并完全开源。如果你不愿信任 Workshop 二进制文件，可以阅读每一行代码或自己从源代码构建 Mod。补丁仅触及特定的 PZ 渲染类——无文件 I/O，无网络访问，PZ 自身代码之外无任何内容。
