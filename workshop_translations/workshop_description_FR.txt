Fini les zombies qui jouent à cache-cache.

Dans PZ vanilla, les murs et les toits ne s'estompent que dans un rayon d'environ 5 tuiles autour de votre personnage — en tant que joueur derrière l'écran, vous voyez donc moins que ce que votre personnage perçoit réellement. Un zombie appuyé contre le mur d'une maison ou caché derrière une haute clôture reste totalement invisible pour vous, même lorsque votre personnage le regarde droit dans les yeux. [b]Peek a View[/b] comble cet écart : les maisons et les clôtures qui bloquent la vue s'estompent de plus loin — ce qui arrive à votre écran correspond (en grande partie) à ce que votre personnage peut réellement voir.

[h1]Ce que fait ce mod[/h1]
[list]
[*]Étend la portée des coupes de ~5 tuiles (vanilla) jusqu'à 20 (configurable).
[*]Se désactive automatiquement au-delà d'une vitesse de conduite configurable, pour garder un FPS stable sur les machines modestes.
[*]Mode optionnel actif uniquement en visée ("nimble stance").
[*]Aucun mur transparent, aucun contournement de la ligne de vue — même mécanique d'estompage qu'en vanilla, simplement déclenchée de plus loin.
[*]Corrige également un bug de rendu en B42 où les structures construites par le joueur font disparaître les murs vanilla à proximité.
[/list]

[h1]Paramètres[/h1]
Toutes les options se trouvent dans Options → Mods → Peek a View. [b]F8[/b] active/désactive le mod (un halo vert/rouge confirme l'état), réassignable sous [noparse][PeekAView][/noparse] dans Options → Raccourcis clavier.

[list]

[h1]Compromis[/h1]
PeekAView modifie volontairement l'expérience visuelle de PZ vanilla — il vaut mieux savoir à quoi vous vous engagez.
[list]
[*]Ce n'est pas le rendu voulu par TIS. La portée étendue des coupes est une rupture délibérée avec le style visuel livré par les développeurs. Si vous préférez le rendu vanilla, ce mod n'est pas pour vous.
[*]Dans les quartiers denses, les bâtiments peuvent s'estomper et réapparaître plus vite lorsque vous vous déplacez — cela peut être distrayant.
[*]Malgré l'optimisation, il y a un léger surcoût par image — surtout visible en conduisant (c'est à ça que sert le seuil de vitesse). À pied, ce n'est en général pas perceptible ; dans les quartiers denses, réduire la portée de quelques tuiles peut aider.
[/list]

[h1]Compatibilité[/h1]

[h1]Pourquoi ZombieBuddy ?[/h1]
Il s'agit d'un mod Java. Modifier le comportement d'une méthode PZ compilée n'est pas réalisable avec un mod Lua classique, et ZombieBuddy fait en sorte que le patch continue de fonctionner à travers les mises à jour mineures de PZ. C'est une installation unique — une fois ZombieBuddy dans votre liste de mods, tout autre mod basé sur ZombieBuddy peut être ajouté sans configuration supplémentaire.

[h1]Code source & transparence[/h1]
PeekAView comme ZombieBuddy sont sous licence MIT et entièrement open source. Si vous préférez ne pas faire confiance au binaire du Workshop, vous pouvez lire chaque ligne ou compiler le mod vous-même depuis les sources. Les patchs ne touchent que certaines classes de rendu de PZ — aucune E/S fichier, aucun accès réseau, rien en dehors du code de PZ lui-même.
